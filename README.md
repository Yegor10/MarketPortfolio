# VIOLETTE — макет онлайн-магазину парфумів

Елегантний макет повноцінного онлайн-магазину нішевої парфумерії у фіолетовому стилі. Чистий стек: HTML + CSS + ванільний JavaScript, без збірки та залежностей. Інтерфейс повністю українською, ціни у гривнях (₴).

## Можливості

- **Головна** (`index.html`) — hero-секція, рухомий рядок переваг, ольфакторні родини, бестселери, промо-банер із таймером зворотного відліку, відгуки, підписка.
- **Каталог** (`catalog.html`) — сітка товарів із фільтрами (родина, для кого, ціна) та сортуванням (популярність, ціна, рейтинг, новинки). Підтримує фільтр з URL: `catalog.html?cat=Східні`.
- **Картка товару** (`product.html`) — галерея, піраміда нот (верх/серце/база), вибір об'єму (30/50/100 мл) з перерахунком ціни, лічильник кількості, схожі аромати. Відкривається за `product.html?id=<id>`.
- **Кошик** (`cart.html`) — зміна кількості, видалення, промокод `VIOLETTE15` (−15%), розрахунок доставки (безкоштовно від 7 000 ₴), оформлення замовлення.
- **Кошик зберігається** у `localStorage` і синхронізується між усіма сторінками (бейдж у шапці).
- Адаптивний дизайн, анімації появи під час скролу, спливаючі сповіщення (toast), мобільне меню.

## Структура

```
MarketPortfolio/
├── index.html      # Головна
├── catalog.html    # Каталог + фільтри
├── product.html    # Сторінка товару
├── cart.html       # Кошик
├── styles.css      # Дизайн-система (фіолетова палітра)
├── script.js       # Дані товарів + логіка кошика
├── images/         # Фото флаконів (по одному на товар, ім'я файлу = id товару)
└── README.md
```

## Запуск

Просто відкрийте `index.html` у браузері. Для коректної роботи навігації можна запустити локальний сервер:

```bash
# Python
python -m http.server 8000
# потім відкрийте http://localhost:8000
```

## Кастомізація

- Кольори та стиль — змінні у `:root` файлу `styles.css`.
- Товари — масив `PRODUCTS` у `script.js`.
- Валюта — функція `fmt` у `script.js` (локаль `uk-UA`, символ `₴`).
- Промокод і вартість доставки — константи у `cart.html`.
