# VIOLETTE — макет онлайн-магазина духов

Элегантный макет полноценного онлайн-магазина нишевой парфюмерии в фиолетовом стиле. Чистый стек: HTML + CSS + ванильный JavaScript, без сборки и зависимостей.

## Возможности

- **Главная** (`index.html`) — hero-секция, бегущая строка преимуществ, ольфакторные семейства, бестселлеры, промо-баннер с таймером обратного отсчёта, отзывы, подписка.
- **Каталог** (`catalog.html`) — сетка товаров с фильтрами (семейство, для кого, цена) и сортировкой (популярность, цена, рейтинг, новинки). Поддерживает фильтр из URL: `catalog.html?cat=Восточные`.
- **Карточка товара** (`product.html`) — галерея, пирамида нот (верх/сердце/база), выбор объёма (30/50/100 мл) с пересчётом цены, счётчик количества, похожие ароматы. Открывается по `product.html?id=<id>`.
- **Корзина** (`cart.html`) — изменение количества, удаление, промокод `VIOLETTE15` (−15%), расчёт доставки (бесплатно от 7 000 ₽), оформление заказа.
- **Корзина хранится** в `localStorage` и синхронизируется между всеми страницами (бейдж в шапке).
- Адаптивный дизайн, анимации появления при скролле, всплывающие уведомления (toast), мобильное меню.

## Структура

```
MarketPortfolio/
├── index.html      # Главная
├── catalog.html    # Каталог + фильтры
├── product.html    # Страница товара
├── cart.html       # Корзина
├── styles.css      # Дизайн-система (фиолетовая палитра)
├── script.js       # Данные товаров + логика корзины
├── images/         # Фото флаконов (по одному на товар, имя файла = id товара)
└── README.md
```

## Запуск

Просто откройте `index.html` в браузере. Для корректной работы навигации можно запустить локальный сервер:

```bash
# Python
python -m http.server 8000
# затем откройте http://localhost:8000
```

## Кастомизация

- Цвета и стиль — переменные в `:root` файла `styles.css`.
- Товары — массив `PRODUCTS` в `script.js`.
- Промокод и стоимость доставки — константы в `cart.html`.
